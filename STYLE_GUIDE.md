# Style Guide — Tax Filer

## Target Users
Small business owners (Ontario incorporations) managing their own taxes. Primarily the owner + a few trusted users. Eventually a paid SaaS product.

## Feeling
Clean, efficient, autonomous. The UI should feel like it does the work — you just confirm.

## Color Palette
- Background base: `#111318` (dark, not pitch black)
- Surface: `#1a1d24`
- Elevated surface: `#21262f`
- Border: `#2a2f3a`
- Text primary: `#e8eaf0`
- Text secondary: `#8b92a5`
- Accent blue: `#3b82f6` (primary actions)
- Accent green: `#10b981` (success, confirmations, positive signals)
- No orange, no purple gradients

## Typography
- System font stack: `-apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif`
- Clean, no decorative fonts
- Minimal use of bold — reserve for labels and key data

## Principles
- Dark mode only
- Minimal chrome — let data breathe
- AI does the work, user confirms
- No placeholders left empty — AI fills them
- Compact, information-dense but not cluttered
